Language Modeling Research

2021

View Paper

Description

My team modified a GPT-2 like transformer model to test how small changes in the attention mechanism affect the performance of the model. The model we based our design on was Hugging Face's PyTorch implementation of GPT-2. Changes were made to the PyTorch source files and perplexity was evaluated after a language-modeling task was completed.

Languages

Python, PyTorch
